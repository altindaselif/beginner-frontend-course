The <pre> Element

      Yıldızlarla süslenmiş gece gökyüzü,

      Ayın ışığıyla aydınlanıyor.

      Kokuduyla sarhoş eden çiçekler,

      Doğanın kollarında huzurla uyuyorlar.

(Paragraftaki boşlukların web sayfasına kodlama sayfası ile birebir şekilde aktarılması isteniyorsa kullanılır. Yukarıdaki örnekte pre yerine p kullanılsaydı tüm yazı yan yana aktarılırdı.)
